# Cadastro de Alunos de Hogwarts
### ` Projeto criado em ORACLE APEX durante o curso Let's Dev da Paipe. `
<br>

## :rocket: A Aplicação:
### Interface para [Hogwarts - Escola de Magia e Bruxaria](https://harrypotter.fandom.com/pt-br/wiki/Escola_de_Magia_e_Bruxaria_de_Hogwarts), que permite acesso ao Relatório de Alunos Cadastrados, sendo possível criar, editar e excluir (CRUD) cadastros através de formulário.  
### :white_check_mark: Tecnologia utilizada:  `Oracle Apex`
